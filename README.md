A personal portfolio website showcasing my art, photography, journals, projects, and more. Each section has its own unique aesthetic and interactive features.

## Built With
- HTML, CSS, and JavaScript
- Features include a photography sound mixer, doodle canvas, animated starry background, and an art gallery
